Graph ILP
==========

This package implements ILP-formulations of multiple NP-hard problems on graphs.

It is organized as follows:

* cuts_flows
  * min_k_flow.py
* matching
  * maxweight.py
* packing
  * max_clique.py 
* partitioning
  * min_vertex_coloring.py (plus test cases)
* examples
  * contains worked out examples for some of the modules
* docs
  * contains files to create a documentation using Sphinx
    1. Add modules to be documented to index.rst
    2. Navigate to the  docs/ directory and run ```make html```
    3. Check the documentation under docs/build
